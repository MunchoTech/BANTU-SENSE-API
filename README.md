### AI-Driven Intent Detection for Nyanja & Bemba

Bantu-Sense is a "Cognitive Justice" project designed to bridge the gap between 
modern AI and local Zambian languages.

## The Architecture
- **The Brain (Python/FastAPI):** Handles NLP and Intent Classification.
- **The Spine (Java/Spring Boot):** Orchestrates data flow and business logic.
- **The Memory (PostgreSQL):** Stores inquiries and user feedback for continuous learning.

## Tech Stack
- **OS:** Arch Linux
- **Database:** PostgreSQL 17
- **Backend:** Java 21, Spring Data JPA
- **Version Control:** Git/GitHub
